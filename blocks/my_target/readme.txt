My Target Block for Moodle

Shows a student's grade for a grade item called 'Target'. You can create this as a manual grade item or any other type of activity as long as the name is 'Target'.

To install, place all files in /blocks/my_target and visit /admin/index.php in your browser.

This block has been tested on the following versions of Moodle: 2.5.1
