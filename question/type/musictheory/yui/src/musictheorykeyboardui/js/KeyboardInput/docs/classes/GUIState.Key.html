getSubsetIDWithinRegister

() Number

Defined in Key.js:61

Returns the zero-based subset ID of the key within the register, with C=0 for white keys and C#/Db=0 for black keys.

For example, the subset ID of G is 4 (i.e. G is the 5th white key above C), and the subset ID of F#/Gb is 2 (F#/Gb is the 3rd black key above C#/Db).

Returns:

Number:

The key's subset ID within the register.
